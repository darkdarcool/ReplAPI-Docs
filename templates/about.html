About

ReplAPI is a module that scraped the internet to find YOUR replit data. It grabs it, and ships the data out to the variable that you are storing it in for your use! Here is how to install the module

Installation

In the shell, use: 
  pip install REPLAPI
Or, in the replit package tab, search REPLAPI to install it via poetry! When you have it installed, you can start using it! Here is an example of how to get a users cycle(upvote) count! 
  import REPLAPI as repl
  cycles = repl.replit_cycles("YourUsername")
  print(f"I have {cycles} cycles!")
This may be a lot to people, so lets break it down.
Of course, we have to import the module to have it in our code. You don't have to call the same thing as us, but we like to because it lets us remeber that when this is called, we are accessign replit data, but it dosen't matter what you call it.
When we have it imported, we have to use it.
Cycles is the variable that we are storing this data in. And then we call the function, are using the import name to specify that we are using data from there, and then we call replit_cycles. This tells us that you are using out function for grabing replit cycles! After this, you see and YourUsername in quotes. YourUsername isn't a person, so don't use that, use your own replit username! The username that you put in tells us what username to look for! If you put any replit username correctly, it will return the cycle count of the username that you put in!
There is a lot of lag for this, because this is utilizing a web crawler and web crawlers that have to read a lot of code and json are pretty slow, tho we are learning new ways to have it go faster, and we are continuing to improve the speed!
When you finish with the basics of ReplAPI, there a bunch more functions to use, that return different data!

Here is a list of all of the functions! 
- replit_langs("username here") [This returns your most used langs]



- replit_post("username here") [This returns the first replit post on your profile list]



- replit_posts("username here") [This returns all of the replit posts on your profile until the load more button]



- replit_bio("username here") [This returns the users bio]



- replit_comment("username here") [This returns the first comment on your replit profile]



- replit_comments("username here") [This returns all of the replit comments on your profie until the load more button]



- replit_user() [This just returns the owner of the repl's username, this has no parameter]



- replit_name("username here") [This returns the nickname of the username]

Sample Projects 

  import REPLAPI
  repl = REPLAPI()
  
  print("This will find your cycles")
  username = input("What is the username? ")
  try:
    cycles = repl.replit_cycles(username)
  except:
    exit("Invalid Username!")
  print(f"{username} has {cycles} cycles!")
